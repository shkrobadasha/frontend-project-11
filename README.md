## Запуск проекта локально

Чтобы запустить проект локально, проделайте следующее:

1. **Клонируйте репозиторий:**

   ```bash
   git clone git@github.com:shkrobadasha/frontend-project-11.git
   ```

2. **Перейдите в папку проекта:**

    ```bash
    cd frontend-project-11
    ```

3. **Установите необходимые зависимости**

   ```bash
   make install
   ````

4. **Запустите проект в режиме разработке**

    ```bash
    make develop
    ```

После этого проект будет доступен по адресу:
```http://localhost:8080``` (или другому порту, если 8080 занят).


5. **Сборка проекта (опциально):**

```bash
make build
```

Собранные файлы будут помещены в папку ```dict```
